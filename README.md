# Adventure Works: Sales & Profitability Audit

## Project Overview
This Power BI dashboard provides a comprehensive audit of sales performance for the fictional company **Adventure Works**. The project transforms raw data into actionable insights focused on financial stability and operational efficiency.

## Key Business Questions Addressed
*   **Financial Health:** Analyzing revenue trends and profit margins across different regions.
*   **Quality Control:** Investigating return rates by product category (identified critical issues in the Canadian market).
*   **Customer Value:** Segmenting customers to identify high-value demographics and purchasing patterns.

## Technical Skills Demonstrated
*   **Data Modeling:** Star Schema design with optimized table relationships.
*   **Advanced DAX:** Created measures for YTD sales, Profit Margins, and Return Rates.
*   **Data Transformation:** Cleaned and reshaped .csv data using Power Query.

## How to View
*   Download the `AdventureWorks_Report.pbix` file to explore the interactive dashboard in Power BI Desktop.
